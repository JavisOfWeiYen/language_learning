# 語言大師 Language Master

一個輕量、簡潔的多語言單字學習工具，支援英文、日文、韓文三種語言獨立學習。所有資料儲存在瀏覽器本地，無需註冊、無需後端。

🔗 **線上版本**：https://javisofweiyen.github.io/language_learning/

---

## 功能特色

### 🌐 多語言切換
- 一鍵切換英文 / 日文 / 韓文，每種語言維護獨立的單字庫、進度與錯題本
- 自動偵測語言並使用對應的瀏覽器語音 (TTS) 朗讀

### 📚 單字管理
- 手動新增單字（含中文翻譯、詞性、例句）
- **批量貼上匯入**：一次貼上多行逗號分隔的文字，即時預覽錯誤與重複
- **CSV 匯入 / 匯出**：與試算表互通，方便備份與分享單字庫
- 全文搜尋、單字熟練度標示

### 🎯 學習測驗
- 聽寫拼字測驗（聽聲音寫單字）
- 聽寫造句測驗（聽聲音寫整句）
- 答錯自動記入錯題本，便於重複複習
- 結合間隔重複 (Spaced Repetition) 原理安排複習時機

### 📊 進度追蹤
- 經驗值與等級制度（入門 → 初階 → 中階 → 進階 → 熟練 → 精通）
- 學習統計：累積單字數、測驗次數、正確率

## 使用方式

### 線上使用
直接打開 [https://javisofweiyen.github.io/language_learning/](https://javisofweiyen.github.io/language_learning/) 即可使用。資料儲存在你的瀏覽器中，清除瀏覽器資料會一併清除單字庫，建議定期匯出 CSV 備份。

### 本地使用
下載 `index.html`，用瀏覽器打開即可。完全離線可用。

---

## 批量匯入格式

每行一筆單字，欄位用逗號分隔（中英文逗號皆可）：

```
單字, 中文, 詞性, 例句
```

範例（韓文）：
```
사과, 蘋果, n., 저는 사과를 좋아해요.
책, 書, n., 이것은 제 책입니다.
가다, 去, v.
```

- 前兩欄（單字、中文）為**必填**
- 後兩欄（詞性、例句）為**選填**
- 系統會自動檢查並標示：可匯入 / 重複 / 缺欄位

## 技術說明

- **單一 HTML 檔案**：所有 CSS、JavaScript 都內嵌於 `index.html`，無外部依賴
- **零後端**：完全在瀏覽器內運作，資料存於 `localStorage`
- **語音功能**：使用瀏覽器內建的 Web Speech API，無需 API 金鑰
- **託管**：GitHub Pages
